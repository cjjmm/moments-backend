# 多媒体展示软件项目技术总结

## 一、项目架构

本项目采用**前后端分离**架构：

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   小程序前端     │────▶│   Spring Boot   │◀────│   后台管理前端   │
│  (微信小程序)    │     │     后端        │     │   (Vue 3 Web)   │
└─────────────────┘     └─────────────────┘     └─────────────────┘
                              │
                              ▼
                        ┌─────────────────┐
                        │     MySQL       │
                        └─────────────────┘
```

---

## 二、后端（Spring Boot）

| 技术 | 版本 | 用途 |
|------|------|------|
| Spring Boot | 2.7.18 | 核心框架 |
| Spring Security | - | 认证授权 |
| MyBatis-Plus | 3.5.3.1 | ORM |
| MySQL | 8.0.33 | 数据库 |
| JWT | 0.9.1 | 令牌认证 |
| Java | 11 | 运行环境 |

**启动方式**：运行 `MomentsBackendApplication.java` 或 `mvn spring-boot:run`

**访问地址**：`http://localhost:8080/api/v1/`

---

## 三、后台管理前端（Vue 3）

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue | 3.5.25 | 前端框架 |
| Vite | 7.2.4 | 构建工具 |
| TypeScript | 5.9 | 类型支持 |
| Element Plus | 2.13.0 | UI 组件库 |
| ECharts | 6.0.0 | 统计图表 |

**目录**：`src/main/resources/static/admin/media-platform-cms`

**启动方式**：
```bash
cd src/main/resources/static/admin/media-platform-cms
npm run dev      # 开发模式 → http://localhost:5173/admin/
npm run build    # 打包后由 Spring Boot 托管 → http://localhost:8080/admin/
```

---

## 四、小程序前端（微信小程序）

| 技术 | 用途 |
|------|------|
| WXML/WXSS/JS | 原生小程序开发 |
| 微信开发者工具 | 开发调试 |

**目录**：`miniprogram/`

**启动方式**：用微信开发者工具导入 `miniprogram` 目录，点击编译

**API 配置**：在 `app.js` 中修改 `baseUrl`
```javascript
baseUrl: 'http://localhost:8080/api/v1'      // 模拟器
baseUrl: 'http://你的IP:8080/api/v1'         // 真机测试
```

**页面结构**：
- `pages/index` - 首页
- `pages/publish` - 发布
- `pages/my` - 个人中心
- `pages/detail` - 详情
- `pages/login` / `register` - 登录注册
- `pages/search` - 搜索
- `components/` - 点赞、评论组件

---

## 五、完整启动流程

| 步骤 | 操作 | 访问地址 |
|------|------|----------|
| 1 | 启动 MySQL | - |
| 2 | 启动 Spring Boot | `http://localhost:8080` |
| 3 | 访问后台管理 | `http://localhost:8080/admin/` |
